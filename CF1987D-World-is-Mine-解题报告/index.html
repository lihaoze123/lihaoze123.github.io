CF1987D World is Mine 解题报告

大致题意

Alice 和 Bob 正在玩一个游戏。初始时有 \(n\) 块蛋糕，第 \(i\) 块蛋糕的美味值为 \(a_i\)。

两人轮流选取蛋糕，Alice 先手：

  • Alice 在自己的回合中，必须选择并吃掉一块剩余蛋糕，其美味值严格大于她之前吃过的所有蛋糕的最大美味值。注意第一回合她可以选择任意蛋糕。
  • Bob 在自己的回合中，可以选择任意剩余蛋糕吃掉。

当当前玩家无法选择符合条件的蛋糕时，游戏结束。设 \(x\) 为Alice吃掉的蛋糕总数。Alice希望最大化 \(x\) ，而Bob希望最小化 \(x\)。

假设双方都采取最优策略，求 Alice 最终能吃掉的蛋糕数量。

分析

对于 Alice，最优策略一定是从小到大依次吃掉剩下的蛋糕。且对于每“种”美味值，她只会吃一次，也即 Bob 要阻止 Alice 吃掉某种美味值 \(x\) 的蛋糕，必须在他自己的回合中吃掉所有美味值为 \(x\) 的蛋糕。

既然 Alice 的最优策略相对固定，我们考虑使用动态规划求解 Bob 的最优策略。

设 \(dp_{i, j}\) 表示对于美味值小于等于 $ i$ 的蛋糕，Bob 吃掉一些组蛋糕后，Alice 能够选择 \(j\) 组蛋糕时，Alice 能吃掉的最小蛋糕数量。

则有以下状态转移： - 对于当前美味值，Bob 不选择吃掉，也即 Alice 能多吃一组，则 \(dp_{i, j + 1} \gets \min(dp_{i, j + 1}, dp_{i - 1, j} + 1)\) - 对于当前美味值，Bob 选择吃掉，则 \(dp_{i, j - cnt_i} \gets \min(dp_{i, j - cnt_i}, dp_{i - 1, j})\)

初始状态为 \(dp_{0, 0} = 0\)，其余为 \(\infty\)。

最终答案为 \(\min_{j \in [0, m]} dp_{m, j}\)，其中 \(m\) 为美味值的最大值。
