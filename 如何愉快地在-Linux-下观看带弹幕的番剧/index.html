如何愉快地在 Linux 下观看带弹幕的番剧

自从之前把主系统换到 Arch Linux 之后，Windows 下的一些常规操作都不能，即使是基本的听歌/追番的体验也很差[1]。

在尝试了半天用 Wine 安装 dandanplay（依赖问题不会解决），使用开源方案 KikoPlay （目前 AUR 上面的版本在 Wayland 显示协议下有谜之窗口分离问题，且目前版本无法添加巴哈姆特的弹幕）之后，我选择了使用脚本从 弹弹play API 获取弹幕，再转换成 mpv 播放器能用的 ASS 字幕格式的方案。

  • 用户ID：字符串形式表示的用户ID，通常为数字，不会包含特殊字符

使用 KikoPlay 播放器观看弹幕

KikoPlay 是基于 libmpv 播放内核的弹幕播放器，具有 WebDav、资料库、aria2 下载 等功能，提供了较为完善的追番体验[2]。

但是在目前阶段还存在一些需要改进的地方：

  • 在 Wayland 显示协议下直接运行会造成主窗口和播放窗口分离

    解决方法是是切换为X11显示协议，或者使用xwayland兼容层来运行。 要强制在Wayland会话中使用X11(需要安装xwayland)，则需要添加“QT_QPA_PLATFORM=xcb”参数。
    同时关闭 进度条预览 选项。

  • 巴哈姆特动画疯的弹幕无法添加

    尚不确定是笔者网络代理的问题，还是播放器自身弹幕下载脚本的问题。由于目前大部分的新番叔叔都没有买，动画疯弹幕的缺失会大大影响弹幕观看体验。

    笔者的解决方案是用脚本把 AcFun 的 json 格式的弹幕转换成 bilibili 的 xml 格式的弹幕，再用 KikoPlay 加载本地弹幕[3]。
